# Loan Approval Prediction

This project predicts whether a loan application will be approved or not using machine learning techniques.

## 📌 Problem Statement
Loan approval is an important decision for financial institutions. The goal of this project is to build a model that can accurately predict loan approval based on applicant information.

## ⚙️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-learn  
- LightGBM  
- Jupyter Notebook  

## 🤖 Machine Learning Approach
- Data preprocessing and cleaning  
- Feature engineering  
- Handling class imbalance using SMOTE  
- Model training using LightGBM  
- Hyperparameter tuning for better performance  

## 📊 Results
- Achieved strong performance with high predictive accuracy  
- Evaluated using ROC-AUC metric  
- Model optimized using cross-validation and tuning techniques  

## 💡 Key Features Used
- Income to loan ratio  
- Employment age ratio  
- Interest rate features  
- Credit history features  

## 🚀 Outcome
This model helps in predicting loan approval decisions efficiently, supporting financial institutions in making data-driven decisions.
